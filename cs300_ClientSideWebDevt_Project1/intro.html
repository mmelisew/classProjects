XHTML rules:

  • Documents must be well formed
  • All element and attribute names must be lower case
  • Non empty elements must have a closing tag
  • Attribute values must always be quoted
  • All tags must nest properly and not overlap
  • Attribute minimization is not allowed

Definition

Element
A pair of HTML tags containing content.
Attribute
Code added to an element to provide more information about the element.
DTD
A set of rules that contains all the elements, attributes and usage rules for the markup language.
URL
It is the unique address of a file's location on the world wide web.
W3C
It sets standards for HTML.
